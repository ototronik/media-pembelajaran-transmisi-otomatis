6. Design Penyampaian Isi Materi Pembelajaran Transmisi Otomatis Berbasis Facebook

Oleh : Isa Muhammad Said (judul skripsi: Pengembangan Media Pembelajaran Transmisi Otomatis Berbasis Facebook)

NIM : 106513405578

Facebook merupakan salah satu media sosial di dunia yang cukup populer dan akrab di Indonesia. Tidak sedikit dari pengguna facebook memanfaatkan media sosial ini sebagai tempat untuk belajar.

Menurut Dian Novitasari: 2006, berdasarkan hasil penelitian dan pengembangan dalam skripsi-nya "pemanfaatan media pembelajaran online interaktif berbasis facebook untuk materi termokimia kelas XI" di peroleh data dan kesimpulan bahwa penggunaan facebook sebagai media pembelajaran layak dan menarik digunakan, dilihat dari hasil validasi kelompok ahli terhadap media yang dikembangkan adalah 100% (valid), dengan hasil validasi isi dari ahli materi (guru dan siswa) dengan presentase masing-masing 82,81% dan 86,6% (valid).

Merujuk dan meninjau hasil penelitian sebelumnya tersebut, peneliti melihat peluang yang ada kemudian melakukan analisis serta percobaan untuk menerapkannya pada mahasiswa Pendidikan Teknik Otomotif Universitas Negeri Malang angkatan tahun 2006. Terdapat sedikit kendala dalam melakukan penerapan konsep pembelajaran tersebut. Seperti, mahasiswa pendidikan teknik otomotif tidak setiap waktu bisa berinteraksi secara online, disebabkan oleh mahasiswa pendidikan teknik otomotif harus menyelesaikan tugas praktikum maupun tugas individu mata kuliah lain selain materi pembelajaran transmisi otomatis.

Untuk lebih mempermudah mahasiswa mendalami materi yang telah disampaikan pada perkuliahan transmisi otomatis, maka peneliti menyediakan sebuah media pembelajaran dalam bentuk CD-Media Pembelajaran Interaktif dimana pengguna CD-media pembelajaran tersebut bisa mengganti dan mengupdatenya sesuai dengan hasil diskusi secara mandiri atau secara bersama sesuai dengan prosedur yang ada.

6.1 Aplikasi Facebook dan Facebook Platform

Aplikasi facebook adalah aplikasi yang tersedia dari facebook untuk penggunanya atau dari dan ke sesama penggunanya. Banyak pengguna facebook yang mengembangkan aplikasi atau membuat aplikasinya sendiri secara independen. Aplikasi facebook terdiri dari beragam bentuk atau bagian, diantarnya aplikasi facebook untuk desktop, aplikasi facebook untuk mobile dan aplikasi facebook atau platform yang dibangun facebook untuk website (social plugin).

CD Interaktif

CD Interaktif merupakan sebuah sarana interaksi yang dapat dibuat oleh siapa saja. Penggunaan CD Interaktif sudah populer dan banyak digunakan dalam dunia pendidikan di Indonesia.

6.2 Mata Kuliah Transmisi Otomatis

Transmisi Otomatis merupakan mata kuliah yang wajib diikuti oleh mahasiswa Jurusan Teknik Mesin Program Studi Pendidikan Teknik Otomotif Univeristas Negeri Malang pada semester ganjil (semester ke-7), dengan jumlah SKS 2 SKS dan jam studi 2 jam dengan 1 jam studi 60 menit. Tujuan dari pembelajaran mata kuliah Transmisi Otomatis ialah agar mahasiswa jurusan teknik mesin khususnya program studi mekanik otomotif dapat mengetahui nama-nama komponen, fungsi dan cara kerja serta dapat menganalisa kerja mekanik, elektronik dan hidrolik. Mampu melakukan overhaul, pemeriksaan dimensi komponen serta penyetelan pada saat uji jalan dan uji kemampuan.

Banyaknya kompetensi yang harus diikuti oleh mahasiswa, secara tidak langsung menimbulkan sebuah kesadaran bagi mahasiswa pendidikan teknik otomotif untuk belajar secara mandiri agar pemenuhan kompetensi tercapai.

6.3 Konsep Penyampaian Media Pembelajaran Berbasis Facebook

Asumsi Penelitian

  1. Mahasiswa memiliki akun facebook
  2. Mahasiswa memiliki CD-Interaktif hasil pengembangan

Ada pun konsep pembelajaran online berbasis facebook yang dikembangkan dapat dideskripsikan pada gambar berikut.

6.4 Deskripsi diagram alur design pengembangan pembelajaran.

Facebook user (mahasiswa) diberikan sebuah CD Interaktif yang didalamnya terdapat file multimedia sebagai sarana pembelajaran yang telah dikembangkan. File CD Interaktif hasil pengembangan yang dibagikan ke mahasiswa merupakan mirror dari website yang diupload di google appengine, website pada (google appengine mempresentasikan gambar dari flickr dan file video dari youtube) . Didalam CD Interaktif terdapat komponen yang dapat digunakan untuk berkomunikasi antara user dan user lain melalui facebook (berkomunikasi publik).

Facebook menyediakan fasilitas berupa javascript SDK yang bisa dipergunakan untuk membuat aplikasi berbasis web dengan menyediakan facebook plugin seperti comment dan send. Kedua komponen ini yang digunakan untuk peneliti dalam mengembangkan media pembelajaran interaktif berbasis facebook.

File dalam bentuk CD Interaktif hasil pengembangan dengan mudah bisa di edit dan dirubah content-nya oleh mahasiswa sendiri. Perubahan isi dari materi pembelajaran transmisi otomatis dilakukan sesuai prosedur yang sediakan pada buku manual dalam CD.

6.5 Kelebihan Media Pembelajaran Hasil Pengembangan

  1. CD media hasil pengembangan bisa digunakan secara offline dan juga online.
  2. Selain melalui media hasil pengembangan, user bisa mengakses media perkuliahan transmisi otomatis melalui website.
  3. Transmisi otomatis merupakan sebuah teknologi dalam dunia otomotif, perkembangan teknologi ini secara bertahap selalu berjalan. Dengan media hasil pengembangan ini, user atau pengguna bisa dengan mudah mengedit isi dalam hasil pengembangan kemudian didiskusikan secara bersama dalam perkuliahan transmisi otomatis kemudian diupdate secara bersama. Media pembelajaran hampir sama dengan konsep open source.
  4. Mempermudah dosen dalam penyampaian materi.
  5. Sebagai solusi untuk menyediakan media pembelajaran yang dibutuhkan oleh mahasiswa pendidikan teknik otomotif berdasarkan tahap analisis yang dilakukan oleh peneliti pada pembahasan BAB 3.

6.6 Kesimpulan

Media pembelajaran berbasis facebook yang berisi media atau multimedia merupakan salah satu hasil perkembangan teknologi informasi dan seni. Dimana informasi ini disampaikan melalui suatu perangkat oleh sistem komputer secara interaktif.

Keterbatasan waktu dan media pembelajaran akan menghambat proses pembelajaran suatu materi, sehingga kompetensi yang harus dicapai tidak terpenuhi. Hal ini dapat dirasakan pada proses pembelajaran mata mata kuliah transmisi otomatis, singkatnya waktu yang harus ditempuh oleh mahasiswa pendidikan teknik otomotif secara tidak langsung memicu mahasiswa untuk mempelajari secara mendalam tentang mata kuliah tersebut.

Keterbatasan media pembelajaran tersebut dapat diatasi dengan penggunaan media pembelajaran dalam proses pembelajaran. Media ini bersifat sebagai media pendamping atau alat bantu pengajar. Keberhasilan media pembelajaran ini tergantung dari beberapa faktor pendukung. Media pembelajaran ini tidak bisa berjalan sendiri tanpa adanya pendukung. Dalam penggunaan media, apabila seseorang pelajar paham dan cakap, maka aktivitas akan berjalan dengan baik dan menguasai bahan pelajaran sedangkan apabila sebaliknya, maka media tersebut bukannya untuk mempermudah bahkan mungkin akan menghambat keberhasilan siswa dalam belajar.

Media pembelajaran ini dibuat untuk memberi solusi kepada mahasiswa sebagai alternatif media pembelajaran, dan mempermudah para pengajar membentuk suatu iklim diskusi pembelajaran dalam perkuliahan transmisi otomatis.

Media pembelajaran ini dinyatakan layak oleh ahli materi, ahli media dan serta user kelompok kecil sebagai media pembelajaran untuk mata kuliah transmisi otomatis berbasis facebook . Hal ini telah dibuktikan bahwa hasil perhitungan menunjukkan :

Hasil analisis pengembangan media pembelajaran berbasis facebook untuk mata kuliah transmisi otomatis pendidikan teknik otomotif fakultas teknik universitas negeri malang - To view this page ensure that Adobe Flash Player version 10.2.0 or greater is installed.

Media pembelajaran berbasis facebook dapat diterapkan dalam belajar mengajar setelah dikatakan layak oleh para ahli dan uji user kelompok kecil. Tahapan selanjutnya adalah media pembelajaran ini akan diuji keefektifannya terhadap peserta didik.

6.7 Prosedur Pengembangan Materi Pembelajaran Transmisi Otomatis

Sesuai dengan pemaparan yang telah disampaikan sebelumnya, para peserta didik dapat menambah atau mengganti konten yang ada dalam CD hasil pengembangan. Berikut ini akan dijabarkan tentang langkah-langkah untuk mengembangkan media pembelajaran transmisi otomatis.

  1. Susun nashkah materi pembelajaran transmisi-otomatis yang akan dikembangkan sesuai dengan buku penulisan karya ilmiah
  2. Diskusikan materi pembelajaran yang telah disusun pada jam perkuliahan atau secara kelompok
  3. Cantumkan refrensi yang anda peroleh
  4. Jika hasil pengembangan telah disetujui oleh dosen pembimbing anda, kemudian bagikan ide atau naskah yang sudah dikembangkan ke peserta didik lain
  5. Pengembang materi pembelajaran transmisi otomatis, tidak wajib mengupdatenya pada website. Namun jika anda berkenan untuk mengembangkan lebih lanjut untuk mengupdate-nya, anda bisa mengirimkan naskah ke pto.ft.um@gmail.com

Format pengembangan

  1. Nama : (berisi nama lengkap anda)
  2. NIM : (berisi NIM anda)
  3. Tahun Angkatan : (berisi Tahun angkatan anda)
  4. Jenis Pengembangan : (Materi isi atau fitur baru yang anda kembangkan)
  5. Naskah : (Naskah yang anda kembangkan)
